# Lina Engine
Lina Engine is an open-source , modular, tiny and fast C++ game engine, aimed to develop 3D cross-platform desktop games.

## Prequisites
Visual Studio 2017

## Installation
- git clone https://github.com/lineupthesky/LinaEngine --recursive
- Open the .sln file
- Run x86 Build
- Open Debug/LinaEngine.exe

## Dependencies
- SDL 2.0
- OpenGL 4.6
- GLEW 2.1.0
- GLM
